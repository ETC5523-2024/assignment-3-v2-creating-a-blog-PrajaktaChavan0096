This is a blog template for the ETC5523 Communicating with Data assessment at Monash University, showcasing my technical projects and skills developed through my academic journey and professional experience.

1 Projects

1.1 Virtual Air Drawing Using Gesture Detection

In this project, I implemented a virtual air drawing tool that uses machine learning algorithms to detect finger movements on camera. The application includes a color palette and uses Python with OpenCV for image processing.

1.2 Hand Gesture Controlled Car

This project involved developing a car that can be controlled using hand gestures, utilizing convolutional neural networks and image processing to track movements and send commands to the car.

1.3 Automated Toll Plaza System

I contributed to a project for automating toll collection using image processing. This project was part of the Texas Instruments Innovation Challenge and helped streamline the toll collection process using camera inputs.
